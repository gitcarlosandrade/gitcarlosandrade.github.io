Data Types

Type Example Description
Number 4, 8, 1516, 23.42 Any number, including numbers with decimals.
BigInt 1234567890123456n Any number greater than 2^53-1 or less than -(2^53-1), with 'n' appended to the number.
String 'hello', "world" Any grouping of characters surrounded by single or double quotes.
Boolean true, false Represents true or false, like an on-off switch.
Null null Represents the intentional absence of a value.
Undefined undefined Represents the absence of a value and indicates that a variable has not been assigned a value.
Symbol Symbol() Unique identifiers used in complex coding.
Object { name: 'John', age: 30 } Collections of related data.
