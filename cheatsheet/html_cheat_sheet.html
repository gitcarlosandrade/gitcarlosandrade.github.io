Semantic Tags

Tag Name Description
<header> Header Defines a header for a document or section.
<footer> Footer Defines a footer for a document or section.
<nav> Navigation Defines a set of navigation links.
<article> Article Defines independent, self-contained content.
<section> Section Defines a section in a document.
<aside> Aside Defines content aside from the content it is placed in.

Non-Semantic Tags

Tag Name Description
<html> HTML Defines the document as HTML.
<head> Head Contains metadata/information for the document.
<title> Title Specifies the document's title.
<body> Body Contains the visible page content.
<h1> to <h6> Headings Defines headers, <h1> being the most important and <h6> the least.
<p> Paragraph Defines a paragraph.
<a> Anchor Defines a hyperlink.
<ul> Unordered List Defines an unordered list.
<ol> Ordered List Defines an ordered list.
<li> List Item Defines a list item.
<img> Image Embeds an image in the document.
<table> Table Defines a table.
<th> Table Header Defines a header cell in a table.
<ol> Ordered List Defines an ordered list.
<li> List Item Defines a list item.
<img> Image Embeds an image in the document.
<table> Table Defines a table.
<th> Table Header Defines a header cell in a table.
<tr> Table Row Defines a row in a table.
<td> Table Data Defines a cell in a table.
<form> Form Defines a form for user input.
<input> Input Defines an input field.
<textarea> Textarea Defines a multiline input field (textarea).
<button> Button Defines a clickable button.
<select> Select Defines a drop-down list.
<option> Option Defines an option in a select list.
<link> Link Defines the relationship between a document and an external resource (commonly used to link to stylesheets).
<meta> Meta Information Defines metadata about the HTML document.
<script> Script Places or references a script, such as JavaScript.
<style> Style Contains style information for a document (CSS).

HTML Tags Examples

Basic Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>

</body>
</html>

Text Formatting

        
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p><b>Bold text</b> and <i>italic text</i>.</p>

Links and Images

        
<a href='https://example.com'>Visit Example.com</a>
<img src='image.jpg' alt='Descriptive text'>

Lists

        
<ul> 
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> 
    <li>First Item</li>
    <li>Second Item</li>
</ol>

Tables

        
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Row 1, Cell 1</td>
        <td>Row 1, Cell 2</td>
    </tr>
</table>

Forms

        
<form action='/submit' method='post'>
    <label for='name'>Name:</label>
    <input type='text' id='name' name='name'><br><br>
    <input type='submit' value='Submit'>
</form>
